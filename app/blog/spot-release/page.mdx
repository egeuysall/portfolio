# Spot v1.0 Launch

**Published: 6 May, 2025**

Spot v1.0 is live.

This one's been a fun build. I made Spot because I was tired of how clunky and irrelevant most event discovery platforms feel. Either you're overwhelmed with options that don't matter, or you're stuck digging through filters just to find something decent.

I wanted to build something simple. Something that actually feels useful and personal.

## What is Spot?

Spot is a web app that helps you discover concerts, meetups, and local events that actually match your interests. No account needed. No clutter. Just answer a few questions, and Spot pulls in a feed of events that makes sense for you.

It's designed to save time and cut through the noise. Instead of dumping every event in your area, it focuses on the ones you'd probably enjoy.

## How It Works

Spot is powered by a pretty lean but solid tech stack:

- Real-time data from the Ticketmaster API
- Personality filtering through the OpenAI API
- Frontend built with Next.js 15, Tailwind CSS, and TypeScript

Everything is tuned for speed and simplicity. The UI is minimal, mobile-first, and fast.

## What’s Next

This is just the starting point. Here’s what I’m thinking for future updates:

- Accounts so you can save preferences and past events
- More advanced filtering (price, distance, category)
- Interactive stuff like event ratings, RSVPs, and comments

The goal is to eventually turn Spot into a space where you don’t just find events but also connect with them in a meaningful way.
